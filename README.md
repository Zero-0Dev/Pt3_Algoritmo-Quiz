# Projeto P3 – Algoritmos e Programação I
**Senac – 2025**

Este repositório contém o projeto prático (P3) desenvolvido para a disciplina Algoritmos e Programação I, orientada pelo professor Guilherme Duarte de Barros.  
O objetivo do trabalho é aplicar conceitos fundamentais de lógica de programação, sintaxe em Java e estruturação de algoritmos, seguindo as orientações fornecidas em sala.

## Tema do Projeto
**Quiz (PI)**  
O programa apresenta perguntas, recebe respostas do usuário e calcula o resultado final, aplicando lógica condicional, laços e manipulação de variáveis.
